# Customer Complaint Management System

## Project Overview
Customer Complaint Management is a Dynamics 365 CRM Model-Driven App used to manage customer complaints and track their resolution process.

## Tables Used

### Contacts
Stores customer information.

Fields:
- Name
- Email
- Mobile

### Complaints
Stores customer complaints.

Fields:
- Complaint Name
- Contact
- Complaint Category
- Description
- Status
- Owner

### Complaint Categories
Stores complaint category master data.

Examples:
- Product Issue
- Service Issue
- Billing Issue
- Technical Issue

## Features

### Contact Management
- Create customer records
- Store email and mobile information

### Complaint Management
- Register complaints
- Assign complaints
- Track complaint status
- Resolve complaints

### Complaint Categories
- Categorize complaints
- Generate reports based on category

## Technologies Used
- Dynamics 365 CRM
- Dataverse
- Model Driven App
- JavaScript
- C# Plugins
